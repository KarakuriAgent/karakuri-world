# Karakuri World - 詳細設計プラン

## 進め方

1. ファイルを上から順に **1つずつ** 作成する（まとめて複数ファイルを進めない）
2. 概要設計（communication-layer.md、world-system.md）を参照しながら詳細を記述する
3. 作成が完了したらチェックボックスに ✅ をつける
4. 1ファイル完了ごとに以下のレビューを実施する
   - `/pr-review-toolkit:review-pr` でレビューを実施
   - `/codex-consultant` でCodexにレビューを依頼
   - 指摘に対して **妥当かどうかを検証** してから修正する（不要な指摘はスキップ）
   - 修正後、再度レビューを繰り返す
   - 指摘がなくなるまで繰り返す
5. レビュー完了後、ユーザーに報告する（次のファイルには自動で進まない）
6. 将来拡張性に関する記述（「将来的に〜」「拡張時には〜」等）は設計負債になり得るため、不要であれば追加しない
